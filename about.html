  • ■当サイトについて

元々これは地獄谷の情報を一つにまとめて、スマートフォンで閲覧できるようにしようと考えて作ったウェブサイトでした。しかし、制作にのめり込んでいくうちにインターネット上に公開したいという想いが強くなりました。その結果がこれです。

(AWS というウェブサービスを使ってサイトを公開し、勝手にアカウントを亡き者にされてポートフォリオが使えなくなったあの日の屈辱を拭い去るためか...)


地獄谷というのは、福島区吉野にある新橋筋商店街裏にある小さな商業地域の名称で、何故そう呼ばれているのかは諸説あります。その珍しい地名に心を打たれ、凸った先は魅力的なお店の数々...流砂の如くめり込んでいきました。


現在インターネットや MAP アプリ等に公開されている情報はどれも不明確かつ不安定、そして古いです。現在進行形で更新されているこのサイトにて、地獄谷というレトロな呑み屋街を知って頂ければと思います。

  • ■セキュリティ等の安全性

当サイトは GitHub 社の GitHub Pages を利用して運営されています。ユーザーのプライバシーとセキュリティを重視しており、GitHub Pages によって提供されるセキュリティ機能を活用しています。


GitHub Pages は、HTTPS（HTTP over SSL）をサポートしており、データの安全性と暗号化通信を確保するために SSL 証明書を自動的に適用しています。これにより、ユーザーの情報が保護され、安全にウェブサイトを閲覧できる環境を提供しています。


ご安心の上、当サイトをご利用ください。

  • ■アクセス解析

当サイトは Google 社の Google Analytics を利用しています。


これらのサービスではユーザーのパソコン・スマートフォンに Cookie を保存し、閲覧行動を分析することができます。


Cookie とはウェブサイトがユーザーのブラウザに保存する小さなテキストファイルのことで、ユーザーの識別情報や行動履歴などを記録するために使用されます。


Google Analytics で得られた情報は趣向や分析などに用いており、個人特定に当たる情報は一切利用しておりません。


安心して当サイトをご利用ください。
